# Web UI для калькулятора податків Freedom24

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Замінити Jupyter notebook простим Flask веб-застосунком з двома сторінками: завантаження файлу та відображення результатів.

**Architecture:** Flask + Jinja2, без бази даних, файл обробляється в пам'яті і не зберігається на диск. Вся бізнес-логіка залишається в `src/` без змін.

**Tech Stack:** Python 3.9+, Flask, Jinja2 (вбудований у Flask), існуючі `src/` модулі.

## Контекст проекту

Існуюча кодова база:
- `src/parser.py` — парсинг Freedom24 JSON
- `src/nbu.py` — курси НБУ
- `src/fifo.py` — FIFO-розрахунок закритих позицій + збагачення UAH

- `src/report.py` — текстова інструкція (залишається для тестів)

Що прибираємо: `tax_calculator.ipynb`, залежності `jupyter`, `notebook`, `ipykernel` з `requirements.txt`.

## Маршрути Flask

| Метод | URL | Опис |
|-------|-----|------|
| GET | `/` | Сторінка завантаження |
| POST | `/calculate` | Обробка файлу → зберігає в session → redirect на `/result` |
| GET | `/result` | Відображення результатів із session |

Патерн: **Post/Redirect/Get** — після POST результати зберігаються у Flask `session`, потім redirect на GET `/result`. Це запобігає повторному відправленню форми при F5.

> **Увага щодо розміру session:** Flask за замовчуванням зберігає session у cookie (~4KB ліміт). Для уникнення переповнення — зберігати в session лише серіалізований JSON рядок з `tax_result` dict та списком `positions`. Якщо розмір все одно перевищує ліміт, використати `flask-session` з `filesystem` backend.

`POST /calculate` приймає:
- `file` — JSON файл Freedom24 (multipart/form-data)
- `tax_year` — рік (int, з форми); допустимий діапазон: 2020–поточний рік; за замовчуванням у `<select>` — попередній календарний рік

**Валідація `tax_year`:** якщо відсутній, не ціле число, або поза діапазоном — повернення на `/` з повідомленням про помилку.

Обробка повністю в пам'яті. У разі будь-якої помилки — повернення на `/` з повідомленням (flash message).

## Послідовність обробки в app.py

```
1. parse_freedom_json(json_data)
       → {trades, dividends, other_income, withholding_taxes}
2. FIFOCalculator(nbu_client).calculate(trades, tax_year)
       → positions (закриті позиції за рік)
3. enrich_dividends_with_uah(dividends, nbu_client)
   enrich_other_income_with_uah(other_income, nbu_client)
   enrich_withholding_taxes_with_uah(withholding_taxes, nbu_client)
       → списки з amount_uah
4. Фільтр dividends/other_income/withholding_taxes за tax_year
5. calculate_taxes(positions, dividends_year, tax_year,
                   other_income=other_income_year,
                   withholding_taxes=withholding_taxes_year)
       → tax_result dict
6. Передати positions + tax_result у шаблон result.html
```

НБУ API викликається в кроках 2 і 3. Якщо недоступний — `requests.RequestException` → повернення на `/` з повідомленням "Не вдалося отримати курси НБУ. Перевірте інтернет-з'єднання."

## Сторінка завантаження (`/`)

**Мова:** українська.

**Структура:**
1. Заголовок: "🇺🇦 Калькулятор податків Freedom24" + підзаголовок
2. Секція "Як отримати звіт у Freedom24" — 4 нумерованих кроки:
   - Крок 1: відкрити freedom24.com, увійти в кабінет
   - Крок 2: Мій рахунок → Звіти → Загальний звіт
   - Крок 3: Формат **JSON**, Період **від дати відкриття рахунку** до **31 грудня** звітного року + червоне попередження про FIFO
   - Крок 4: завантажити файл нижче та вказати звітний рік
3. Форма:
   - File input (accept=".json")
   - Select: звітний рік (поточний рік і два попередніх)
   - Кнопка "Розрахувати →"
4. Якщо є помилка — червоний банер під формою

## Сторінка результатів (`/calculate`)

**Структура зверху вниз:**

### 1. Дисклеймер (жовтий банер)
```
⚠️ Цей інструмент надається виключно для довідки. Перевірте розрахунки
самостійно або з податковим консультантом перед поданням декларації.
Автор не несе відповідальності за помилки.
```

### 2. Підсумкова смуга (зелена)
| ПДФО | ВЗ | До сплати |
|------|----|-----------|
| X грн | X грн | **X грн** |

### 3. Два стовпці

**Лівий стовпець — "Розрахунок"** (звідки суми):

Три картки з таблицями:
- **Дивіденди**: сума доходу / ПДФО (9% eff.) / ВЗ
- **Cashback (scores_rebate)**: сума доходу / ПДФО (18%) / ВЗ
- **Торгівля ЦП**: кількість закритих позицій / чистий прибуток / ПДФО / ВЗ; якщо 0 позицій — "Додаток Ф1 не потрібен"

**Правий стовпець — "Що вписати в декларацію"** (таблиця поле→значення):

- **Крок 1 — Розділ II, рядок 10.10** (завжди показується якщо є іноземні доходи):

  | Поле | Значення |
  |------|---------|
  | r01010g2s | США, долар США (USD) |
  | r01010g3 сума | X грн |
  | r01010g6 ПДФО | X грн |
  | r01010g7 ВЗ | X грн |

  Підпис: "↳ для довідки: дивіденди X + cashback X"

- **Крок 2 — Рядок 10.8** (якщо є чистий прибуток від торгівлі):

  | Поле | Значення |
  |------|---------|
  | r0108g3 прибуток | X грн |
  | r0108g6 ПДФО | X грн |
  | r0108g7 ВЗ | X грн |

  Якщо прибутку немає — зелена картка "✓ Рядок 10.8 — не заповнюється"

- **Крок 3 — Додаток Ф1**:
  - Якщо є угоди → "Заповни по таблиці нижче"
  - Якщо немає → зелена картка "✓ Ф1 не подається — продажів не було"

### 4. Таблиця Ф1 (тільки якщо є закриті позиції)

## Обробка помилок

Всі помилки повертають на `/` через Flask `flash()`. Окремий шаблон помилки не потрібен.

| Ситуація | Повідомлення |
|----------|-------------|
| Файл не обрано | "Оберіть файл для завантаження" |
| Файл не є валідним JSON | "Невірний формат файлу. Завантажте JSON звіт з Freedom24" |
| JSON є валідним, але структура не відповідає Freedom24 | "Файл не схожий на звіт Freedom24. Перевірте що обрали правильний файл" |
| НБУ API недоступний | "Не вдалося отримати курси НБУ. Перевірте інтернет-з'єднання" |
| `tax_year` невалідний | "Невірний звітний рік" |

**Порожній звіт** (нема угод і дивідендів): показати результати з нулями. Всі секції відображаються з `0.00 грн`. Таблиця Ф1 — прихована (немає угод). Кроки в правій колонці показують нулі з позначкою "Ф1 не подається".
