# Калькулятор податків Freedom24

Локальний інструмент для розрахунку ПДФО та ВЗ з виписки Freedom24 брокера.
Допомагає заповнити рядок 10.10 та Додаток Ф1 декларації (cabinet.tax.gov.ua).

## Повний флоу декларування (порядок важливий)

> ⚠️ **ВИМОГА: спочатку замов довідку F1419104, потім формуй звіт.**
> Без неї ти не побачиш рядки **10.13 / 11.3** (ознаки 125 / 126 / 127 / 160)
> і отримаєш камералку від ДПС. Freedom24 покриває тільки рядок 10.10 —
> решту дає лише довідка з ДРФО.

1. **Замов довідку F1419104** (ДО запуску `app.py`).
   ЕК `cabinet.tax.gov.ua` → "Введення звітності" → подай запит формою
   **F1401803** ("Запит про суми виплачених доходів…"). Період — звітний рік,
   тип — для декларування. Підписати КЕП, надіслати. Протягом доби у
   "Вхідних документах" прийде довідка **F1419104** — у ній усі доходи з
   кодами ознак (101, 125, 126, 127, 160…) по кожному податковому агенту.

   > Навіщо: агенти (банки, роботодавець) бачать лише свої виплати. Саме
   > через коди **125 / 126 / 127 / 160** цього року ДПС масово розсилає
   > камералку — вони йдуть у рядки 10.13 і 11.3, яких Freedom24-звіт не
   > містить взагалі.

2. **Звіти з Freedom24** (іноземний брокер — у ДРФО його не буде).
   Деталі — у розділі ["Як отримати файл з Freedom24"](#як-отримати-файл-з-freedom24)
   нижче. Треба JSON (для цього скрипта) + той самий звіт у PDF (у пакет
   документів до ДПС).

3. **Прогнати JSON через `app.py`** + ввести підсумки з F1419104 у
   опційній секції форми (ознаки 126 / 127 з ПДФО/ВЗ утриманими агентами,
   ознаки 125 / 160 без податку). На виході отримаєш готові значення для
   рядків **10.10, 10.13, 11.3** і таблиці Додатка Ф1.

4. **Заповнити декларацію в ЕК**, звіряючись:
   - рядки 10.1–10.9, 10.11–10.12 → **довідка F1419104** (крок 1)
   - рядок **10.10** → результат `app.py` (крок 3, Freedom24)
   - рядки **10.13 / 11.3** → результат `app.py` (крок 3, F1419104)
   - Додаток Ф1 додавати **тільки якщо є операції з Freedom24** (є дані у
     таблиці Ф1 на екрані результату). Порожній Ф1 — помилка.

## Вимоги

Потрібен **Python 3.9+**. Перевірити чи встановлений:

```bash
python3 --version
```

Якщо не встановлений:

- **macOS:** встанови через [Homebrew](https://brew.sh) — `brew install python` або завантаж з [python.org](https://www.python.org/downloads/)
- **Windows:** завантаж інсталятор з [python.org](https://www.python.org/downloads/) → при встановленні постав галочку **"Add Python to PATH"**
- **Linux (Ubuntu/Debian):** `sudo apt install python3 python3-venv`

## Запуск

```bash
python -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

Відкрий http://localhost:8080 у браузері.

За замовчуванням порт **8080**. Щоб змінити:

```bash
PORT=5000 python app.py
```

> На macOS порт 5000 може бути зайнятий AirPlay Receiver (System Settings → General → AirDrop & Handoff → AirPlay Receiver → вимкнути).

## Як отримати файл з Freedom24

1. Відкрий freedom24.com → Мій рахунок → Звіти → Загальний звіт
2. Формат: **JSON**
3. Період: **від дати відкриття рахунку** до 31 грудня звітного року
   > Повна історія потрібна для правильного FIFO-розрахунку
4. Додатково сформуй той самий звіт у **PDF** — прикладати як підтверджуючий
   документ, якщо ДПС надішле запит до рядка 10.10.

## Дисклеймер

або з податковим консультантом. Автор не несе відповідальності за помилки.
